# Day 20 - Architecture Proposal

> _December 20, 2025 - Architecture Proposal_

![snippet of the day](img/day20.png)

  ==> **Loading Fractured Memory** <==

On the 20th day, the memory is throwing you into a heated meeting between elves and Santa.

They don't agree on how they should handle their new building infrastructure...
The idea is to power the new building with magical power instead of electricity.

But the equipment used is from the Sleigh system using Reindeer to passively power it.

It seems the architecture target of Santa isn't making everyone happy.

**You have to help them**

## Challenge ‚Äî Propose an architecture to handle this new feature.

Today the exercise is not a code one. Not strictly speaking.
You are to analyze the code base that you probably know, the sleigh system.

The technical experts are clear on how reindeer would need to power it.

### Technical report from the manufacturers
 - 20 magical units per minute would be sufficient to power a big building.
 - Each reindeer would need to rest at least 10 minutes between shift to replenish their power.

You need to come up with a new architecture to handle both the sleigh AND the building.

You can come up with a mermaid graph to explain your changes.
